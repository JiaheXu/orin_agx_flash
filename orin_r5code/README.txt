This package contains the rt-aux-cpu-demo-fsp sources which provide a FreeRTOS
based software demo to run Real-Time applications on the Cortex-R5 Sensor
Processing Engine (SPE) on the Jetson platforms.

How to get started:
Follow rt-aux-cpu-demo-fsp/doc/welcome.md and rt-aux-cpu-demo-fsp/doc/compile-flash.md for the details.
